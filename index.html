SynchroAir.

The Adaptive Fan Controller.

It doesn't just run; it thinks. Seamlessly integrates real-time environmental data with user goals to deliver optimal, energy-efficient comfort.

Discover the Intelligence

Intelligence Engineered.

Predictive AI: Time to Cool

The fan doesn't guess. It uses an Exponential Moving Average (EMA) model to observe the live rate of temperature change, and based on this data, accurately predicts the remaining "Time to Cool" (TTC) to your target. This provides intelligent foresight, moving beyond simple automation.

Predictive Modeling on Arduino Uno.

Adaptive P-Control Loop

The system utilizes a Proportional (P) Control algorithm. It constantly calculates the difference (Error) between the current temperature and your target. The fan's PWM output is adjusted proportionally, ensuring the speed is always optimized—never too fast, never too slow.

Closed-Loop Feedback System.

Sensing the Environment.

BME280 Precision

Acquires Temperature, Humidity, and Pressure via the I2C bus. Provides the foundational data necessary for the P-Control loop and the humidity-aware comfort calculation.

Encoder Input

The Rotary Encoder provides a clean, intuitive interface for setting the target temperature and toggling between Active Cooling and Target Setup modes. Utilizes hardware interrupts (D2/D3).

Clear I2C Feedback

The 16x2 I2C LCD displays all crucial system variables, including the current TTC prediction, fan status (Low/Med/High), and real-time sensor readings.

Power Engineered for Safety.

Robust Power Isolation (TIP31C)

The 12V DC fan is an inductive load, requiring careful power handling. We use the **TIP31C NPN transistor** to create a low-side switch, completely isolating the high-current fan circuit from the low-voltage Arduino logic. PWM from Pin D9 controls the transistor base (via 1kΩ resistor) safely.

Inductive Spike Protection

When the fan is switched OFF, the motor generates a damaging high-voltage spike (inductive kickback). A "1N4007 Flyback Diode" is installed in reverse-bias across the fan terminals, safely clamping and dissipating this energy, preventing immediate destruction of the TIP31C transistor.
